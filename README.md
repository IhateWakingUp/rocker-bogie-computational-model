# rocker-bogie-computational-model
Computational Modeling and dynamic analysis of a modified Rocker Bogie suspension using Matlab and Msc Adams . 
The kinematic  model has been developed in Matlab for understanding how the suspension works . 
For the dynamic Model ,  we have created and analyzed the CAD model of the UGV in Msc Adams.
